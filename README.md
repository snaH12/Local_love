テーマを選んだ理由

　地方にずっと住んでいて、当たり前だと思っていたことが全然知られていないことがあり、ぜひ知ってほしいと思いました。
　また、旅行でいろいろなところへ行ってみたいという夢があり、旅行した時にその地元の人がおすすめするスポットを巡ってみたいと思い、このテーマにしました。

ターゲットユーザ

　旅行をしてみたいと思ったことのある人

主な利用シーン

　旅行をするきっかけや旅行気分を味わいたいとき、地元の情報を知ってほしいと思った時
